# gcal_sync

特定のカレンダーのスケジュールを、別のカレンダーに同期するアプリ。

## 動作概要
以下、コピー元となるカレンダーをFROM, コピー先となるカレンダーをTOとする。

1. FROMから、一定期間のスケジュール情報を取得
2. TOの、同期間のスケジュールを削除
3. 1.で取得したスケジュールの開始・終了時間・タイトルと同内容の予定をTOに登録

## 利用方法

同期を実行

```shell
make
```

この`gcal_sync`で作成された予定を削除。全てのカレンダーから削除する。

```shell
make clear
```

同期用の認証情報を作成

```shell
make credentials
```
